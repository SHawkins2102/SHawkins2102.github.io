Background Color

The "background-color" property is used to change the color of the website.

With CSS, a color is mainly specified by: a valid color name, a hex value, or an RGB value.

body {
background-color: lightblue;
}
  • a valid color name - like "lightblue"
    • Background Image

    The "background-image" property is for specifying which image to use as the background for an element.

    When using a background image, make sure to use an image that does not disturb the text.

    body {
    background-image: url("paper.gif");
    }
  • Note: The image might disturb the text if not used properly.
    • Background Repeat

    The "background-repeat" property repeats an image both vertically and horizontally.

    Showing the background image only once is also specified when using the "background-repeat" property.

    body {
    background-image: url("gradient_bg.png");
    }
  • If the image above is repeated only horizontally, use (background-repeat: repeat-x;), the background will look better.
    • Background Attachment

    The "background-attachment" property specifies whether the background image should scroll or be fixed (will not scroll with the rest of the page)

    When making a background attachment, make sure to specify that the image is not only fixed to the page, but is also able to scroll alongside the page.

    body {
    background-image: url("img_tree.png");

    background-repeat: no-repeat;

    background-position: right top;

    background-attachment: fixed;
    }
  • It's very important that you specify which background image is being fixed in this situation.
    • Background Position

    The "shorthand" property helps shorten the code, while also specifying all the background properties into one single property.

    It does not matter if one of the property values is missing, as long as the other ones are in this order.

    body { background: #ffffff url("img_tree.png") no-repeat right top; }
  • The shorthand property not only helps position lines of code, but also shortens them as well for your convenience.
